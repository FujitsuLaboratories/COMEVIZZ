Introduction

COMEVIZZ

comevizz

comevizz is web application to analyze distributions of source code metrics. You can see the statistical information of many projects and Z-Score of specific projects.

See comevizz for more details.

sonar2csv

sonar2csv is CLI tool to translate metrics data measured by SONARQUBE Software to CSV format. This CSV formatted file can be used for input for comevizz.

See sonar2csv for more details.
