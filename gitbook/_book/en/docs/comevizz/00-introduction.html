Introduction

comevizz

What is comevizz?

comevizz is a web application to analyze source code metrics distributions. You can see the statistical information for many projects on comevizz. You can also see Z-Score for metrics of your own project, and know the room for improvement in your source code.

See comevizz for more details.

comevizz uses .csv file for metrics data measured by SONARQUBE Software™ 1. We provide CLI tool sonar2csv to obtain metrics data in .csv format from SONARQUBE Software.

See sonar2csv for more details.

What statistics can you find?

  • Z-score
  • Cumulative distribution function
  • Probabilistic density function
